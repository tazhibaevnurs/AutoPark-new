Настоящая Политика определяет порядок и условия обработки персональных данных пользователей сайта AUTOPARK в соответствии с требованиями законодательства Российской Федерации.

1. Принципы обработки данных

  • законность и справедливая основа обработки;
  • обработка данных только для заранее определенных целей;
  • соразмерность объема данных целям обработки;
  • достоверность и актуальность обрабатываемых сведений;
  • ограничение срока хранения данных.

2. Категории обрабатываемых данных

Оператор может обрабатывать: имя, телефон, адрес электронной почты, текст обращений в формах сайта, а также технические данные (IP-адрес, cookie, параметры браузера и сведения о сессии).

3. Цели обработки

  • прием и обработка заявок;
  • обратная связь с пользователями;
  • повышение качества сервиса и поддержки;
  • выполнение требований законодательства.

4. Правовые основания обработки

Правовыми основаниями являются добровольное согласие пользователя, выраженное при заполнении форм сайта, а также нормы действующего законодательства РФ.

5. Порядок хранения и защиты

Доступ к персональным данным ограничен. Используются организационные и технические меры для предотвращения несанкционированного доступа, изменения, блокирования, копирования или распространения данных.

6. Передача персональных данных

Передача возможна только в объеме, необходимом для обработки обращений, либо в случаях, прямо предусмотренных законодательством РФ.

7. Права субъекта персональных данных

Пользователь вправе получить информацию об обработке данных, потребовать их уточнения, блокирования или удаления, а также отозвать согласие на обработку.

8. Контакты по вопросам персональных данных

По вопросам обработки персональных данных можно обратиться через страницу контактов или по телефону +7 (961) 163-41-57.
